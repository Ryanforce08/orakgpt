HTML Project 6

2 points - Programming Project

Create a folder named HTML6 (should be inside of your HTML folder)

Directions:
1. Open up Notepad ++ from your USB Drive.
2. At the top, you will see a tab for Language. Go to H → HTML
3. Go to Save As… and locate the directory you created before Step 1 and type in index.

Your goal is to create a website of your choice using multiple concepts learned in HTML so far:
1. You must include CSS
2. You must have a change of background
3. You must have a change of fonts (color, face / family, size, etc)
4. You must have a border
5. You must have a picture
